Autostar applications


To autostart applications in X create *.desktop files
under ~/.config/autostart/:

1. Create ~/.config/autostart
$ cd ~
$ mkdir -p .config/autostart
$ cd ~/.config/autostart/

To autolaunch xfce4-terminal:
$ cp /usr/share/applications/xfce4-terminal.desktop .
